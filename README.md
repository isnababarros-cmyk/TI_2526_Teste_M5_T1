## Exercícios

1. Explica o que é uma fórmula numa folha de cálculo. Indica dois exemplos de operações que podem ser feitas com fórmulas. (3 valores)

        Resposta:Uma fórmula é uma instrução utilizada numa folha de cálculo para realizar cálculos automáticos com valores das células. Sempre que os dados mudam, o resultado da fórmula atualiza-se automaticamente. Com fórmulas é possível, por exemplo, somar valores (=A1+B1) ou multiplicar valores (=A1*B1).

2. Define “referência relativa” e “referência absoluta”. Dá um exemplo de cada. (3 valores)

        Resposta: A referência relativa adapta-se automaticamente quando a fórmula é copiada para outra célula, como por exemplo A1. A referência absoluta mantém-se fixa, mesmo quando a fórmula é copiada, sendo indicada com o símbolo $, como por exemplo $A$1.

3. Na célula D2 encontra-se a fórmula:

    =$B$2*C2

    Que tipo de referência é usada em B2? O que acontece à referência C2 quando a fórmula é copiada para D3? (2 valores)

        Resposta: Em B2 é usada uma referência absoluta. Quando a fórmula é copiada para D3, a referência C2, que é relativa, passa automaticamente para C3.

4. Para que serve a formatação condicional? Dá dois exemplos práticos. (3 valores)

        Resposta:A formatação condicional serve para alterar automaticamente o aspeto das células de acordo com determinadas regras. Por exemplo, pode ser usada para destacar valores negativos a vermelho ou para colorir notas positivas a verde.

5. Explica o que é um filtro de dados e dá um exemplo prático da sua utilização. (2 valores)

        Resposta:Um filtro de dados permite mostrar apenas os dados que cumprem um determinado critério, escondendo os restantes. Por exemplo, filtrar uma lista de alunos para mostrar apenas os que tiveram positiva.

6. Indica a diferença entre “ordenar” e “filtrar” dados. (2 valores)

        Resposta: Ordenar significa organizar os dados segundo uma ordem específica, como crescente ou alfabética. Filtrar significa mostrar apenas os dados que cumprem uma condição, ocultando os outros.

7. Para que serve um gráfico numa folha de cálculo? Indica um tipo de gráfico e uma situação em que possa ser utilizado. (3 valores)

        Resposta: Um gráfico serve para representar os dados de forma visual, facilitando a sua análise. Um exemplo é o gráfico de colunas, que pode ser utilizado para comparar vendas mensais de uma empresa.

8. Escreve a fórmula necessária para calcular a prestação mensal de um empréstimo a 12 meses com taxa anual de 3% sobre 500€. Não é necessário calcular o valor final. (2 valores)

        Resposta: PGO(3%/12;12;500)
